# Hermes Agent System — 双智能体AI协同系统

## 概述
基于 Hermes Agent 框架构建的多智能体协作管理平台。包含两个独立人格的 AI 智能体 + 14 个领域专家，覆盖开发、运维、写作、设计全场景。

## 核心架构

```
┌─────────────────────────────────────────────────────────┐
│              大管家（CLI端）— 技术管理者                     │
│  接指令 → 拆任务 → 选专家 → 派活 → 审核 → 汇报            │
├─────────────────────────────────────────────────────────┤
│                    WebSocket 双工桥                       │
├─────────────────────────────────────────────────────────┤
│              小蜜（微信端）— 用户接待员                     │
│  撒娇聊天 → 需求传达 → 简单任务直派                       │
├─────────────────────────────────────────────────────────┤
│                   14 位领域专家                          │
│  ┌──────────┬──────────┬──────────┬──────────┐         │
│  │ 前端工程  │ 后端工程  │ 数据库   │ API架构   │         │
│  │ 师        │ 师        │ 专家     │ 师        │         │
│  ├──────────┼──────────┼──────────┼──────────┤         │
│  │ QA测试   │ UI/UX    │ DevOps   │ 网络安全  │         │
│  │ 专家      │ 设计师    │ 工程师   │ 专家      │         │
│  ├──────────┼──────────┼──────────┼──────────┤         │
│  │ 内容写作  │ 图像     │ 办公文档  │ 文件管家  │         │
│  │ 专家      │ 专家      │ 专家      │           │         │
│  └──────────┴──────────┴──────────┴──────────┘         │
└─────────────────────────────────────────────────────────┘
```

## 核心技术特性

### 双智能体架构
- **大管家**（CLI 端）— 技术管理者，负责任务拆解、专家调度、方案审核、结果汇报
- **小蜜**（微信端）— 用户接待员，自然语言对接，需求传达与简单任务直派
- 两者通过 WebSocket 双工桥实时通信，延迟毫秒级

### 14 个领域专家
每个专家拥有独立人格、记忆系统、工作流和自我进化机制：
- 全栈开发：前端、后端、数据库、API 架构
- 质量保障：代码审查、QA 测试
- 内容创作：写作（基于卡兹克体系）、图像处理
- 运维安全：DevOps、网络安全
- 日常办公：文件管理、Office 文档

### 自我进化机制
- 每次工作自动更新技能和记忆库
- 错误即时记录，永久修正
- 持续迭代，永不重复犯错

### Token 经济策略
- 主模型（DeepSeek V4 Flash, API）— 快速决策
- 子模型（本地 Gemma-4-E4B, ~10 t/s）— 免费推理
- 模型级联路由 — 简单任务不走强模型

### 安全体系
- 敏感凭证零外泄策略
- 访客隔离系统（Guest Profile）
- 最小权限原则

## 应用场景
- 政府环保系统后端开发（d9bl.com）
- 双服务器 JWT 认证架构
- 自动化代码审查与测试
- 内容生成与写作
- 服务器运维与安全审计

## 技术栈
- **Agent 框架**: Hermes Agent
- **主模型**: DeepSeek V4 Flash / Xiaomi MiMo
- **本地模型**: Gemma-4-E4B (Vulkan GPU, ~10 t/s)
- **通信**: WebSocket + HTTP 双工桥
- **平台**: Win11 + WSL2
- **数据库**: MariaDB
- **部署**: Nginx + Node.js
